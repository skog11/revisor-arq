# MCP Pendientes de Evaluación

## Supabase MCP
- **Estado**: No disponible como paquete oficial
- **Alternativa**: Utilizar REST API de Supabase directamente desde el código
- **Prioridad**: Alta - simplificaría operaciones de base de datos y autenticación
- **Investigación requerida**: Verificar si existe un MCP comunitario o crear uno personalizado

## Otros MCPs potencialmente útiles
- **Vercel MCP**: Para despliegues y gestión de variables de entorno
- **GitHub MCP**: Ya disponible - utilizar para gestión de repositorios y pull requests
- **Filesystem MCP**: Ya disponible - acceso mejorado al sistema de archivos local
